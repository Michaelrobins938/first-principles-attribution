## Recording Checklist

### Before Recording

- [ ] Frontend running on port 3002
- [ ] Backend running on port 8000
- [ ] Sample data ready
- [ ] Browser cache cleared
- [ ] Screen recording software ready (OBS, QuickTime, etc.)

### Technical Setup

```bash
# Terminal 1: Backend
cd backend
python -m uvicorn main:app --reload --port 8000

# Terminal 2: Frontend  
cd frontend
npm run dev
```

### Recording Tips

1. **Full Screen**: Record at 1920x1080 minimum
2. **Cursor**: Hide cursor during uploads
3. **Timing**: Practice the transitions beforehand
4. **Audio**: Record in a quiet environment
5. **Editing**: Keep cuts clean, no transitions needed

### Export Settings

- **Format**: MP4
- **Resolution**: 1920x1080
- **Frame Rate**: 30fps
- **Bitrate**: 15-20 Mbps
